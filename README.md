# EcoWasteClassifier_MIPHI_project_practice
EcoWasteClassifier — система классификации отходов с нейросетью MobileNetV2. Модель определяет тип мусора по изображению среди 6 классов: cardboard, glass, metal, paper, plastic, trash. Проект полностью реализован в Google Colab и включает тестирование на новых изображениях.
